Overview

This module provides a secure way to hash and verify passwords using the Argon2 password hashing algorithm. It includes rate-limiting functionality for the hashing process to prevent abuse. Users can register by creating a username and password, and then log in by verifying their credentials.

Features

  • Secure password hashing using Argon2.
  • Rate limiting for hashing passwords to enchance security.
  • Simple user registration and login simulation.

Requirements

Here are requirements needed to use Bishash.

  • Python 3.x
  • argon2-cffi library
  • ratelimit library

Installation

pip install argon2-cffi ratelimit
