Overview

This module provides a secure way to hash and verify passwords using the Argon2 algorithm with rate-limiting functionality to prevent abuse.

Features

  • Secure password hashing using Argon2
  • Rate limiting for enhanced security
  • Simple user registration and login simulation

Requirements

  • Python 3.x
  • argon2-cffi library
  • ratelimit library

Installation

pip install argon2-cffi ratelimit
