API Reference

Here are the endpoints you'll need to interact with the Bishash:

  • POST /v1/charges - Create a new charge.
  • GET /v1/customers - Retrieve customer details.
  • POST /v1/tokens - Generate a new token.

Examples

Below is a code example to create a charge:

curl https://example.com/v1/charges \
  -u sk_test_xxxx: \
  -d amount=2000 \
  -d currency=usd \
  -d source=tok_visa \
  -d description="Charge for test@example.com"
